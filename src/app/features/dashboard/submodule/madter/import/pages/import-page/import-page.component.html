Importar workspaces

Carga un Excel con la estructura de workspaces, tableros y tareas. El sistema crea todo automáticamente en tu cuenta.

¿Cómo funciona?

1
Descarga la plantilla

Haz clic en "Descargar plantilla" para obtener el Excel con el formato correcto y ejemplos.

2
Llena el Excel

Cada fila es una tarea. Agrupa por workspace y tablero. La hoja "Instrucciones" explica cada columna.

3
Carga el archivo

Arrastra el Excel aquí o haz clic para seleccionarlo. Soporta .xlsx y .xls.

4
Revisa el resultado

Verás cuántos workspaces, tableros y tareas se crearon, y los errores por fila si los hay.
